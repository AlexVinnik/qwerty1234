Главное окно¶

На главном окне расположены две таблицы:

  • Таблица текущих проектов (таб.1)

  • Таблица сотрудников участвующих в проекте (таб.2)

Инструкции для каждой из таблиц представлены далее.

Таблица текущих проектов (таб.1)¶

Таблица отображает текущие проекты компании, для этой таблицы существует несколько кнопок, функционал которых описан ниже:

Добавление¶

  1. Нажмите на кнопку добавить, что бы добавить новый проект (будет открыто новое окно)

  2. Введите номер договора, выберите группу и руководителя проекта

  3. После создания, нажмите кнопку «ОК»

Изменение¶

  1. Нажмите на тот проект который хотите изменить

  2. После нажмите кнопку «Изменить» (будет открыто новое окно)

  3. В окне измените данные тех параметров, которые вам нужны

  4. Нажмите кнопку «ОК» и ваши изменения будут применены
